Campos Requeridos y Tipos

Al realizar una reserva, asegúrate de completar los siguientes campos obligatorios según el tipo de evento:

  • Nombre y Contacto: Campo de texto para el nombre y teléfono del organizador.
  • Fecha del Evento: Selector de fecha para elegir la fecha del evento.
  • Tipo de Evento: Botones de opción para seleccionar entre banquete, jornada o congreso.
  • Número de Personas: Campo de texto para indicar la cantidad de asistentes. Límites: 50 para jornadas y congresos, 100 para banquetes.
  • Tipo de Cocina: Lista desplegable para elegir entre buffet (vegetariano o no) o carta.

Recuerda que algunos campos, como el número de días y habitaciones, estarán activos solo para eventos de tipo congreso.
